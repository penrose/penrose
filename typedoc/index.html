@penrose/core

@penrose/core

@penrose/core contains the core of the Penrose platform, including the compilation pipeline and the SVG renderer.

See the website for more details about the system in general, and see @penrose/components for React components wrapping around this core library.

Getting started

If using Penrose in a web or Node application, run yarn add @penrose/core to install the package.

core contains some browser-only dependencies (e.g. MathJax for rendering equations). If using Node, we recommend using global-jsdom to shim them.

import "global-jsdom/register"; // must come before the Penrose import
import * as Penrose from "@penrose/core";

Use JavaScript modules to import functions from @penrose/core. Use either a CDN that supports ES modules (e.g. JSPM) or a bundler (e.g. vite). For example, run the following to set up starter project with vite:

yarn create vite
cd vite-project
yarn add @penrose/core
yarn add -D vite-plugin-top-level-await # `@penrose/optimizer` uses top-level await and this vite plugin adds support for that

In vite.config.js:

import { defineConfig } from "vite";
import topLevelAwait from "vite-plugin-top-level-await";

export default defineConfig({ plugins: [topLevelAwait()] });

In index.html:

<!DOCTYPE html>
<html>
  <body>
    <div id="penrose-diagram"></div>
    <script type="module" src="/main.js"></script>
  </body>
</html>

In main.js:

import * as Penrose from "@penrose/core";
Penrose.diagram(
  {
    substance: `
  Set A, B
  IsSubset(A, B)
  `,
    style: `
  canvas {
      width = 400
      height = 400
  }
  forall Set s {
      s.shape = Circle {}
      ensure lessThan(20, s.shape.r)
  }
  forall Set s1, s2
  where IsSubset(s1, s2) {
      ensure contains(s2.shape, s1.shape)
      s2.shape above s1.shape
  }
  `,
    domain: `type Set
  predicate IsSubset(Set, Set)`,
  },
  document.getElementById("penrose-diagram")
);

Finally, run the following to see the output:

yarn dev
